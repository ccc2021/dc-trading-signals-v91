> 給訂閱用戶的完整使用教學。所有功能皆在 Telegram 完成。

## 🚀 第一次使用

1. 打開 Telegram，搜尋 Bot 用戶名（向管理員索取或從邀請連結進入）

3. 看到主選單後，按需求點擊或輸入指令

## ⚙️ 個人設定 `/settings`

### 🔔 通知設定 `/notify`

可分項開關：

- 進場訊號
- 止盈通知 (TP1/TP2/TP3)
- 止損通知 (SL)
- 訊號更新（移動止損等）
- 每日績效報告
- 系統公告
- 行情警報

### 🌙 安靜時段 `/quiet`

設定不打擾時段（例如 23:00–07:00）。安靜時段內的訊號**不會丟失**，會在結束後補送。

### 💰 資金設定 `/capital` `/risk`

```
/capital 50000   # 設定交易資金 USD 50,000
/risk 1.5        # 單筆風險 1.5%
```

設定後，每張訊號卡會顯示 **建議口數**（依品種 tick value 自動換算）。

### 🌍 時區設定 `/timezone`

```
/timezone Asia/Taipei
/timezone America/New_York
/timezone Europe/London
```

影響「安靜時段」的判斷與時間顯示。

## 🆕 v9.2 新功能

### 📐 計算機（即時、不用會員）

```
/rr 21500 21480 21540          # 算 R:R = 1:2
/size NQ 21500 21480           # 依你的資金/風險算建議口數
```

`/size` 會用你的 `/capital` 與 `/risk` 設定，搭配品種 tick value 自動算。

### 🛡️ 風控進階

```
/setlimit 50      # 每日虧損上限 50 點，達上限後訊號卡會印警示
/setlimit 0       # 取消上限
/setbe on         # TP1 達成時系統建議移動止損到成本
/setmax 1         # 同品種最多同時 1 張未平倉，超過時警示
/img on           # 訊號改用圖片版（含 entry/sl/tp 圖表）
/img off          # 回純文字
```

> 警示是**提醒**，不會阻擋訊號發送。

## 💡 小技巧

- **訊號錯過了？** 用 `/signals` 把今天訊號拉回來看
- **不想被半夜吵？** `/quiet` 啟用安靜時段，醒來再補看
- **品種太多？** `/subscribe` 只勾你想交易的，免被洗版
- **不會算口數？** `/capital` + `/risk` 設一下，訊號會自動算
- **要換時區？** `/timezone Asia/Tokyo` 即可

需要協助時，按主選單的「📞 聯繫客服」或輸入 `/contact`。
