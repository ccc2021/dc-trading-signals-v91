用戶透過 Telegram 就能：
✅ 選擇要接收哪些品種的訊號
✅ 設定訊號類型偏好（短線/波段）
✅ 設定安靜時段（不打擾）
✅ 自訂通知方式
✅ 管理個人資金與風險
✅ 查看個人化績效統計
✅ 一鍵續費/升級

```sql
  subscribed_symbols TEXT DEFAULT '["NQ","ES"]',

  -- 時區
  timezone TEXT DEFAULT 'Asia/Taipei',
  
  -- 語言
  language TEXT DEFAULT 'zh-TW'
);
```
